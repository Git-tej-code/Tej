# Integration Failures

1. **API call fails**

**Cause**

* Incorrect endpoint
* Invalid credentials
* Network issues

**Resolution**

* Test integration directly
* Validate credentials
* Check request preview and logs

&#x20;

2. **Integration works in test but fails in process**

**Cause**

* Environment mismatch
* Missing input mapping

**Resolution**

* Verify environment context
* Ensure runtime inputs are passed correctly
