# Locked or Archived Items

1. **Cannot edit a Process / Activity / Agent**

**Cause**

* Item is locked
* Item is archived

**Resolution**

* Create a new version
* Unlock only if governance allows
* Archived items cannot be edited

&#x20;

2. **Cannot delete an item**

**Cause**

* Deletion is intentionally not supported
* AutoX uses archiving for governance

**Resolution**

* Archive the item instead
* Use naming conventions to indicate deprecation
