# Common Error & Messages

1. **“You do not have permission to perform this action”**

Cause

* Role does not include the required permission
* ABAC scope restriction (Department / Function / Environment)

**Resolution**

* Verify your assigned role
* Verify you are operating in the correct scope
* Contact Platform Admin for access review

&#x20;

2. **“Invalid configuration” or “Validation failed”**

Cause

* Missing required parameters
* Incorrect input/output mapping
* Unsupported data type

**Resolution**

* Open the canvas (Process / Activity / Agent)
* Review highlighted validation errors
* Ensure all mandatory fields are populated
