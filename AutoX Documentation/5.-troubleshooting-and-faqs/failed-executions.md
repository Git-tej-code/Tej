# Failed Executions

1. **Process fails during execution**

**Cause**

* Task misconfiguration
* Missing inputs
* Integration failure

**Resolution**

* Review execution logs
* Inspect trace in Agent Observer
* Validate task input/output mapping

&#x20;

2. **Activity runs but produces no output**

**Cause**

* Output not mapped
* Downstream task expects different data type

**Resolution**

* Review task output configuration
* Check downstream input mapping
