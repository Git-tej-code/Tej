# General

1. **What makes AutoX different from RPA or BPM tools?**

**AutoX combines:**

* Workflow orchestration
* AI agents
* Observability
* Governance-first design

This enables intelligent automation with accountability.

&#x20;

2. **Can I directly edit production workflows?**

**No.**

**AutoX enforces:**

* Versioning
* Package-based deployment
* Environment isolation

This prevents accidental production changes.
