# Access Denied Issues

1. **Cannot see Departments / Functions / Processes**

**Cause**

* ABAC restriction on organization or department
* Role has read-only or no access

**Resolution**

* Confirm department assignment
* Confirm role permissions
* Switch to correct organization or environment

&#x20;

2. **Cannot access Agent Builder or Agent Observer**

**Cause**

* Platform-specific access not granted

**Resolution**

* Agent Builder and Agent Observer require explicit access
* Ask Platform Admin to grant platform access
