**AutoX is composed of three tightly integrated platforms.**

**1. Main Platform**

Used for:

* Orchestration (Departments, Functions, Processes)
* Workflow design (Processes, Activities, Tasks)
* Integrations
* Package management and deployment
* User and role management

**2. Agent Builder**

Used for:

* Designing AI agents visually
* Connecting models, data, tools, and logic
* Testing agents safely in Playground
* Exporting and sharing agents via APIs or MCP

**3. Agent Observer**

Used for:

* Monitoring execution traces
* Analyzing cost and latency
* Evaluating agent quality
* Auditing behavior and system usage

Each platform has independent access control and may or may not be available to every
