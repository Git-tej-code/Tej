# Navigation Overview

**The left navigation in AutoX is role-aware.**

Common navigation areas include:

* Dashboard
* Departments
* Functions
* Processes
* Scheduler
* Agent Builder
* Agent Observer
* Integrations
* User & Roles
* Version Control
* Settings
* Notifications
* Profile

**You may not see all items, depending on:**

* Your role
* Your organization
* Your environment (Dev / QA / UAT)

This is intentional.
