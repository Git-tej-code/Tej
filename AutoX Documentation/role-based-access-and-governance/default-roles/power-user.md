# Power User

**Role Overview**

Power Users are advanced operational users responsible for building and running workflows.

&#x20;

**Capabilities**

* Create, edit, and manage:
* Departments
* Functions
* Processes
* Activities
* Tasks
* Agents (via Agent Builder)
* Execute processes
* View observability data

&#x20;

**Restrictions**

* Cannot create or delete users
* Cannot manage global settings
* Deployment permission may be restricted

&#x20;
