# Develop A Process(Canvas)

**Who can do this**

* Platform Admin
* Power User
* Integration Engineer

&#x20;

**When to Use Develop**

Use Develop when:

* Adding execution logic
* Connecting Activities
* Configuring integrations or agents

&#x20;

**Steps**

1. Open the Process
2. Click Develop
3. From the left panel, drag:
4. Start
5. End
6. Activity
7. Agent Activity
8. Sub-Process (if required)
9. Connect components logically
10. Click Save

&#x20;

**What Happens Next**

* Execution logic is mapped
* Process remains editable until locked

&#x20;

**Important Notes**

* Develop view enforces valid connections
* Activities referenced must be active
* Validation errors appear inline
