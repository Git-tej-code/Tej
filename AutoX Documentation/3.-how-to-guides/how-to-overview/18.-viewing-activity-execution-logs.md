# Viewing Activity Execution Logs

**Who can do this**

* Platform Admin
* Power User
* Integration Engineer

&#x20;

**Steps**

1. Open the Activity
2. Click Logs
3. Review:
4. Execution steps
5. Task outputs
6. Errors (if any)

&#x20;

**Important Notes**

* Logs are immutable
* Logs are linked to trace IDs
* Use Agent Observer for deep analysis
