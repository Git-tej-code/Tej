# Access Agent Builder

**Who can do this**

* Agent Developer
* Power User
* Platform Admin

&#x20;

**Steps**

1. Log in to the AutoX Main Platform
2. From the left navigation, click Agent Builder
3. If prompted, log in to the Agent Builder workspace

&#x20;

**What Happens Next**

* You land on the Agent Builder Home Screen
* Only agents and projects you are authorized for are visible

&#x20;

**Important Notes**

* Agent Builder access is role-based
* Some users may see Agent Builder but have read-only access
