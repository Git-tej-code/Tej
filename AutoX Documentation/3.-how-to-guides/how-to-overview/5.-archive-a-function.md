# Archive a Function

**Archiving a Function**

**Steps**

1. Navigate to Functions
2. Locate the function
3. Click Archive
4. Confirm action

&#x20;

**What Happens Next**

* Archived functions:
* Cannot accept new processes
* Remain visible for audit and reporting
* Existing packages and deployments are unaffected

&#x20;

**Important Notes**

* Functions cannot be permanently deleted
* Archiving is reversible
* Audit logs capture all changes
