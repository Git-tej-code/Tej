# Create an Activity

**Who can do this**

* Platform Admin
* Power User
* Integration Engineer

Business Users may view activities but cannot create them.

&#x20;

**Prerequisites**

* Parent Process must exist
* Process version must be Unlocked
* You must have access to the associated Function

&#x20;

**Steps**

1. Open the Process
2. Navigate to Activities
3. Click Create Activity
4. Enter:
5. Activity Name
6. Description
7. Click Create

&#x20;

**What Happens Next**

* A new Activity Version (v1.0) is created
* Activity status is set to In Progress
* Activity opens in the Activity Canvas

&#x20;

**Important Notes**

* Activities are reusable across processes
* Activities cannot be deleted
* Every edit creates a new version automatically
