# Testing an Integration

**Who can do this**

* Platform Admin
* Integration Engineer

&#x20;

**Steps**

1. Open the Integration
2. Select an Operation
3. Click Test
4. Provide test input values
5. Run the test

&#x20;

**What Happens Next**

* Test request is executed
* Response and errors are displayed
* No process execution is triggered

&#x20;

**Important Notes**

* Tests do not affect production workflows
* Failed tests help validate configuration
