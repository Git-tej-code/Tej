# Use Agent Tasks

**Who can do this**

* Platform Admin
* Power User

&#x20;

**What is an Agent Task?**

An Agent Task invokes an AI agent built in Agent Builder.

Use cases:

* Decision-making
* Classification
* Summarization
* Conversational logic

&#x20;

**Steps**

1. Drag Agent Task into the activity
2. Select an available Agent
3. Configure:
4. Input mapping
5. Output handling
6. Connect to the workflow
7. Click Save

&#x20;

**What Happens Next**

* Agent executes as part of the activity
* Agent trace is generated in Agent Observer

&#x20;

**Important Notes**

* Only approved agents can be used
* Agent versions are locked at execution time
* Costs are tracked per invocation
