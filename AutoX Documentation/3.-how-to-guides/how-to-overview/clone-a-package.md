# Clone a Package

**Who can do this**

* Platform Admin
* Power User (with deployment permission)

&#x20;

**When to Clone a Package**

* You want to modify an existing package
* You want to test changes safely
* The existing package is locked

&#x20;

**Steps**

1. Open Version Control â†’ Packages
2. Locate the source package
3. Click Clone
4. Provide:
5. New Package Name (or keep same)
6. Description (optional)
7. Click Confirm

&#x20;

**What Happens Next**

* A new package version is created
* All process mappings are copied
* New package is editable

&#x20;

**Important Notes**

* Cloning preserves version integrity
* Original package remains unchanged
* This is the recommended way to work
