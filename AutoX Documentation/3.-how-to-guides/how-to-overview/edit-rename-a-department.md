# Edit / Rename a Department

**Who can do this**

* Platform Admin
* Power User

&#x20;

**Steps**

1. Navigate to Departments
2. Locate the department
3. Click Edit
4. Update:
5. Department Name
6. Description
7. Click Save

&#x20;

**What Happens Next**

* Changes are applied immediately
* Existing Functions and Processes remain unaffected

&#x20;

**Important Notes**

* Renaming a department does not change ownership or access history
* Audit logs capture the change
