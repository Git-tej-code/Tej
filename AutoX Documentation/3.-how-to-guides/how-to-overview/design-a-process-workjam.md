# Design A Process(Workjam)

**Who can do this**

* Platform Admin
* Power User
* Business User (design-only)

&#x20;

**When to Use WorkJam**

**Use WorkJam when:**

* Designing business flow
* Collaborating with non-technical users
* Reviewing logic without execution details

&#x20;

**Steps**

1. Open the desired Process
2. Click WorkJam Board
3. Add:
4. Start
5. Activities
6. Decisions
7. End
8. Arrange flow visually
9. Click Save

&#x20;

**What Happens Next**

* Logical flow is saved
* No execution logic is added yet
* Process remains In Progress

&#x20;

**Important Notes**

* WorkJam does not execute logic
* It is safe for business review
* Changes do not affect locked versions
