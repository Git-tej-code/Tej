# Edit Activity Logic

**Who can do this**

* Platform Admin
* Power User
* Integration Engineer

&#x20;

**Steps**

1. Open the Activity
2. Click Develop Activity
3. From the left panel, drag required Tasks onto the canvas
4. Connect tasks in logical order
5. Configure each task as required
6. Click Save

&#x20;

**What Happens Next**

* Execution logic is updated
* Activity remains editable until locked
* Validation errors (if any) are shown inline

&#x20;

**Important Notes**

* Task connections must respect data flow
* Invalid connections are not allowed
* Activity logic executes sequentially unless configured otherwise
