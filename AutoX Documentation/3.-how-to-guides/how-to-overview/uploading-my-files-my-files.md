# Uploading My Files ( My Files )

**Steps**

1. Open My Files
2. Upload supported files
3. Add File Component to the canvas
4. Select uploaded file

&#x20;

**Supported Formats**

* Images: png, jpg, jpeg
* Documents: supported only via file components (not Playground)
