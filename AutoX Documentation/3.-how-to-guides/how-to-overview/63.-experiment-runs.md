# 63. Experiment Runs

**What Are Experiment Runs?**

Experiments involve running new prompt or model versions against a versioned Dataset of inputs and expected outputs.

Experiment runs allow:

* Comparing prompts
* Testing model changes
* Evaluating agent improvements safely

&#x20;

**Steps**

1. Navigate to Experiments
2. Select dataset/import dataset
3. Run experiment
4. Compare results

&#x20;

**Important Notes**

* Experiments do not affect production
* Results are fully traceable
