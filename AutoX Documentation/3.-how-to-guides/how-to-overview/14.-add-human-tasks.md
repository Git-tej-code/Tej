# Add Human Tasks

**Who can do this**

* Platform Admin
* Power User

&#x20;

**When to Use a Human Task**

Use a Human Task when:

* Manual approval is required
* Human decision-making is necessary
* Business validation is needed before continuation

&#x20;

**Steps**

1. Open the Activity Canvas
2. Drag Human Task from the task palette
3. Configure:
4. Task name
5. Instructions for the human
6. Expected output
7. Connect it to the workflow
8. Click Save

&#x20;

**What Happens Next**

* Process pauses at the Human Task
* Assigned user receives notification
* Workflow resumes after human action

&#x20;

**Important Notes**

* Human Tasks create execution checkpoints
* SLAs and escalation can be configured (if enabled)
* All actions are logged for audit
