# Understanding Agent Builder Home Screen

**What You See on Entry**

**The Agent Builder home screen includes:**

* Projects\
  Folder-like grouping of agents
* Create New Agent\
  Starts a new agent workflow
* Search Agent\
  Search across all agents in the project
* Ready Agent List\
  Prebuilt or reusable agents
* MCP Servers\
  Entry point to configure MCP servers

&#x20;

**Important Notes**

* There is currently no global MCP server list outside this screen
* MCP servers can also be discovered inside the canvas search
