# 62. Analyse Cost & Model Usage

**Steps**

1. Navigate to **Home** or **Dashboard**
2. View:

* Model Costs
* Cost by Model
* Cost by Type

3. Drill down by:

* Time
* User
* Trace

&#x20;

**Metrics Available**

* Prompt tokens
* Completion tokens
* Total tokens
* Cost per model
* Cost per request

\
**How to Track Spend**

1. Total Cost: Check the "Total Cost" widget to see the aggregate spend ($) for your selected timeframe.
2. Model Breakdown: Analyze the Model Costs widget to identify spending drivers:

* Compare Models: See the cost difference between models (e.g., gpt-4o vs. gpt-4.1).
* Token Volume: Verify the volume of tokens consumed.
* Unit Economics: Track the exact USD value per model.

3. Analyze Trends: Use the Model Usage charts to spot unusual spikes in consumption that may indicate inefficiencies or high traffic.

&#x20;

**Important Notes**

* Cost is calculated per execution
* Costs are immutable and auditable
* Agent Builder testing and production are tracked separately
