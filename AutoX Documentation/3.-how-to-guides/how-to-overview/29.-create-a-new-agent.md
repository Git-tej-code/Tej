# 29. Create a New Agent

**Who can do this**

* Agent Developer
* Power User

&#x20;

Steps

1. New Agent: From your Project dashboard, click "New Agent".
2. Templates vs. Blank Canvas:
3. Templates: Start with a pre-configured setup like "Basic Chatbot," "Vector Store RAG," or "Multi-Agent Orchestrator."
4. Blank: Start from scratch for complete control.
5. Defining the Logic:
6. Drag a Chat Input component to the canvas.
7. Add a Model (e.g., OpenAI, Groq) and an Agent component.
8. Connect the Chat Input to the Agent.
9. Add a Chat Output component to display the final response.

&#x20;

What Happens Next

* Agent opens in the Agent Canvas
* Initial version is created automatically

&#x20;

Important Notes

* Blank is recommended for advanced users
* Templates accelerate common use cases
