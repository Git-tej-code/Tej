# Create a New Agent

**Who can do this**

* Agent Developer
* Power User

&#x20;

**Steps**

1. From the Agent Builder home screen, click Create New Agent
2. Choose one of the following:
3. Template
4. Use Case
5. Methodology
6. Blank
7. Enter:
8. Agent Name
9. Description (optional)
10. Click Create

&#x20;

**What Happens Next**

* Agent opens in the Agent Canvas
* Initial version is created automatically

&#x20;

**Important Notes**

* Blank is recommended for advanced users
* Templates accelerate common use cases
