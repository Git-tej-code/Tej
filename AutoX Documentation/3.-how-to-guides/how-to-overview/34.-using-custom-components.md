# 34. Using Custom Components

**Creating a Custom Component**

1. Drag a "Custom Component" block from the sidebar.
2. Click the Code icon on the node to open the code editor.
3. Define your class inheriting from Component.
4. Specify inputs and outputs using the Agent Builder SDK.

**Important Notes**

* Uploaded components appear under Saved
* Custom components behave like native ones
