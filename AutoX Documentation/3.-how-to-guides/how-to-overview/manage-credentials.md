# Manage Credentials

**Who can do this**

* Platform Admin

&#x20;

**Steps**

1. Navigate to Integrations
2. Open the Credentials tab
3. Click Add Credential
4. Provide:
5. Credential Name
6. Credential Type
7. Secret values
8. Click Save

&#x20;

**What Happens Next**

* Credentials are encrypted and stored securely
* Credentials become reusable across integrations

&#x20;

**Important Notes**

* Credentials are never exposed in plaintext
* Updating credentials does not change integration logic
* Credential usage is audited
