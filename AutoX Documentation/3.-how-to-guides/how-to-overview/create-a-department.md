# Create a Department

**Who can do this**

* Platform Admin
* Power User

Business Users and Agent Developers cannot create Departments.

&#x20;

**Prerequisites**

* You must be logged in to the Main Platform
* You must have access to the Organization where the department will be created

&#x20;

&#x20;

**Steps**

1. From the left navigation, click Departments
2. Click Create Department
3. Enter the following details:
4. Department Name
5. Description (optional but recommended)
6. Click Save

&#x20;

**Important Notes**

* Departments cannot be permanently deleted
* Departments can be archived if no longer required
* Department name should reflect a real business unit
