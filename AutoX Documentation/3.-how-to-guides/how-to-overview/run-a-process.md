# Run a Process

**Who can do this**

* Platform Admin
* Power User

&#x20;

**Prerequisites**

* Process version must be Unlocked
* All referenced activities must be valid

&#x20;

**Steps**

1. Open the Process
2. Click Run
3. Provide required inputs (if prompted)
4. Start execution

&#x20;

**What Happens Next**

* Process execution begins
* Execution logs are generated
* Trace is available in Agent Observer

&#x20;

**Important Notes**

* Running a process does not deploy it
* Execution failures do not affect versions
* Use Agent Observer for debugging
