# Use Connector Tasks

**Who can do this**

* Platform Admin
* Power User
* Integration Engineer

&#x20;

**When to Use Connector Tasks**

Use Connector Tasks to:

* Call external APIs
* Send or receive data
* Integrate third-party systems

&#x20;

**Steps**

1. Drag a Connector Task into the activity
2. Select an existing Integration
3. Configure:
4. Operation
5. Request parameters
6. Headers (if required)
7. Map inputs and outputs
8. Click Save

&#x20;

**What Happens Next**

* Task invokes external system at runtime
* Responses are captured and logged

&#x20;

**Important Notes**

* Credentials are managed centrally
* Connector tasks cannot store secrets
* Failures surface in execution logs
