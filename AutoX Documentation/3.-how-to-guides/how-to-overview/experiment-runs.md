# Experiment Runs

**What Are Experiment Runs?**

Experiment runs allow:

* Comparing prompts
* Testing model changes
* Evaluating agent improvements safely

&#x20;

**Steps**

1. Navigate to Experiments
2. Select dataset
3. Run experiment
4. Compare results

&#x20;

**Important Notes**

* Experiments do not affect production
* Results are fully traceable
