# Version an Activity

**Who can do this**

* Platform Admin
* Power User

&#x20;

**When to Create a New Version**

* You need to change logic
* Existing version is locked
* You want to experiment safely

&#x20;

**Steps**

1. Open the Activity
2. Click Add Version
3. Provide optional version notes
4. New version opens in In Progress state

&#x20;

**What Happens Next**

* Previous versions remain unchanged
* New version can be edited freely
* Versions can be switched via dropdown

&#x20;

**Important Notes**

* Versioning is automatic and immutable
* Version history is always preserved
