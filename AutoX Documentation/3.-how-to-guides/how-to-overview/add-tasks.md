# Add Tasks

**Who can do this**

* Platform Admin
* Power User
* Integration Engineer

&#x20;

**Prerequisites**

* Activity must exist
* Activity version must be Unlocked
* You must be in the Activity Canvas

&#x20;

**Steps**

1. Open the Activity
2. Click Develop Activity
3. From the left task panel, drag a Task onto the canvas
4. Place the task at the required position
5. Connect it to the preceding and succeeding tasks
6. Click Save

&#x20;

**What Happens Next**

* Task becomes part of the execution flow
* Task executes when the activity runs
* Validation is performed on save

&#x20;

**Important Notes**

* Tasks cannot exist outside activities
* Execution order is defined by connections
* Invalid connections are blocked
