# Super Admin

**Role Overview**

The Super Admin operates at the organization layer and governs the platform itself.

This role is intentionally limited in operational usage and focused on platform control.

&#x20;

**Capabilities**

* Onboard new Organizations
* Onboard Platform Admins
* Manage:
* Organization records
* Billing
* Global auditing
* View:
* Number of organizations
* Platform admins
* Usage metrics (high level)

&#x20;

**Restrictions**

* No access to:
* Dashboards
* Processes
* Agents
* Observability data

Super Admin governs the platform, not workflows.
