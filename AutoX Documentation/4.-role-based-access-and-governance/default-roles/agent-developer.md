# Agent Developer

**Role Overview**

Agent Developers focus exclusively on AI agent design and testing.

&#x20;

**Capabilities**

* Full access to:
* Agent Builder
* Agent Playground
* MCP configuration (within Agent Builder)
* View:
* Agent Observer (if granted)

&#x20;

**Restrictions**

* Read-only access to:
* Departments
* Functions
* Processes
* Cannot deploy packages
* Cannot manage integrations or users
