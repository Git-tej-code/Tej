# Platform Admin

**Role Overview**

The Platform Admin owns the AutoX platform configuration and governance within an organization.

&#x20;

**Capabilities**

* Manage users and roles
* Full access to:
* Departments
* Functions
* Processes
* Activities
* Tasks
* Integrations
* Agent Builder
* Agent Observer
* Version Control
* Deployments
* View:
* Audit logs
* System performance

&#x20;

**Restrictions**

* Cannot perform Super Admin actions
* Cannot manage billing across organizations
