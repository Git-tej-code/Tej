# Archive a Department

**Who can do this**

* Platform Admin
* Power User

&#x20;

**Prerequisites**

* Department must exist and be Active

&#x20;

**Steps**

1. Navigate to Functions
2. Click Create Function
3. Select:
4. Department
5. Enter:
6. Function Name
7. Description
8. Click Save

&#x20;

**What Happens Next**

* Function is created under the selected Department
* A default Package is automatically created
* Processes can now be created under this Function

&#x20;

**Important Notes**

* Functions represent business capabilities, not technical modules
* Naming should reflect operational purpose (e.g., â€œInvoice Processingâ€)
