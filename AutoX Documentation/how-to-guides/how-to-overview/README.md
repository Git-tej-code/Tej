# How-To Overview

The How-To Guides provide step-by-step, practical instructions for using AutoX to build, run, and manage automation. Each guide focuses on a single task or outcome and is written to support daily, hands-on usage of the platform.

This section is the primary working reference for builders, power users, and administrators.

**In this section, you will learn how to:**

* Create and manage departments and functions
* Design and develop processes and activities
* Use different task types, including agents and integrations
* Build and test agents using Agent Builder
* Package, deploy, and observe automation safely
