# Understanding Home vs Dashboard

**Home (Default View)**

Home provides a collective overview across all traces.

It includes:

* Request volume
* Cost trends
* Score trends
* Latency metrics

&#x20;

**Dashboard**

Dashboard is a focused, purpose-built view.

You can:

* Create custom dashboards
* Track specific KPIs
* Monitor experiments or releases

&#x20;

**Important Notes**

* Home cannot be deleted
* Dashboards can be created, analyzed, or deleted
