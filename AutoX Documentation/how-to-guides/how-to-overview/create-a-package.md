# Create a Package

**Who can do this**

* Platform Admin
* Power User (if deployment permission is granted)

&#x20;

**Prerequisites**

* At least one Function must exist
* You must have deployment permission for the Function
* Processes must already exist (can be empty initially)

&#x20;

**Steps**

1. From the left navigation, click Version Control
2. Navigate to Packages
3. Click Create Package
4. Enter:
5. Package Name
6. Associated Function
7. Description (recommended)
8. Click Save

&#x20;

**What Happens Next**

* A new package is created
* Package versioning starts automatically
* Package is available in Dev state by default

&#x20;

**Important Notes**

* Packages cannot be deleted
* Package names can remain the same across versions
* Creation does not deploy anything
