# Lock/Unlock An Activity

**Who can do this**

* Platform Admin
* Power User

&#x20;

**Steps to Lock**

1. Open the Activity
2. Verify the correct version
3. Click Lock
4. Confirm action

&#x20;

**Steps to Unlock (If Allowed)**

1. Open the Activity
2. Click Unlock
3. Confirm action

&#x20;

**What Happens Next**

* Locked activities:
* Cannot be edited
* Can be used by processes
* Unlocking may be restricted by governance

&#x20;

**Important Notes**

* Locked activities ensure execution consistency
* Locking does not deploy the activity
* Audit logs record lock actions
