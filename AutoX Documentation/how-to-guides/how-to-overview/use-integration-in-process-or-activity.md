# Use Integration in Process or Activity

**Who can do this**

* Platform Admin
* Power User
* Integration Engineer

&#x20;

**Steps**

1. Open the target Activity
2. Drag a Connector Task onto the canvas
3. Select the desired Integration
4. Choose the required Operation
5. Map:
6. Input parameters
7. Output fields
8. Click Save

&#x20;

**What Happens Next**

* Integration is invoked at runtime
* Response data flows to downstream tasks

&#x20;

**Important Notes**

* Connector Tasks do not store credentials
* Integration availability depends on role and scope
* Failures appear in Activity Logs and Agent Observer
