# Configure Task Inputs & Outputs

**Who can do this**

* Platform Admin
* Power User
* Integration Engineer

&#x20;

**Steps**

1. Select the Task on the canvas
2. Open Task Properties
3. Configure:
4. Input parameters
5. Output parameters
6. Map inputs from:
7. Previous task outputs
8. Static values
9. Global variables
10. Click Save

&#x20;

**What Happens Next**

* Task receives inputs at runtime
* Outputs become available to downstream tasks

&#x20;

**Important Notes**

* Input/output data types must match
* Invalid mappings cause execution failure
* Outputs are traceable in logs
