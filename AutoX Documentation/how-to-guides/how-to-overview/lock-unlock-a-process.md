# Lock/Unlock a Process

**Who can do this**

* Platform Admin
* Power User

&#x20;

**When to Lock a Process**

* Design is complete
* Ready for packaging or deployment
* No further edits required

&#x20;

**Steps to Lock**

1. Open the Process
2. Verify you are on the correct Version
3. Click Lock
4. Confirm action

&#x20;

**Steps to Unlock (If Allowed)**

1. Open the Process
2. Click Unlock
3. Confirm action

Unlocking may be restricted by governance rules.

&#x20;

**What Happens Next**

* Locked processes:
* Cannot be edited
* Can be deployed via package
* New changes require creating a new version
