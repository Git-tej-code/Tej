# Test Agent In Playground

**Who can do this**

* Agent Developer
* Power User

&#x20;

**Steps**

1. Open the agent
2. Click Playground
3. Enter chat input or voice input
4. Run the agent

&#x20;

**Playground Features**

* Chat history stored under Sessions
* Rename sessions
* Delete sessions
* Upload images for testing (png, jpg, jpeg)

&#x20;

**Important Notes**

* Playground does not support PDF, DOC, XLS testing
* Each run creates a session trace
