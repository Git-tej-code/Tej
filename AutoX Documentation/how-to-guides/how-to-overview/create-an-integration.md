# Create an Integration

**Who can do this**

* Platform Admin
* Integration Engineer
* Power User

Business Users and Agent Developers cannot create integrations.

&#x20;

**Prerequisites**

* You must have access to the Main Platform
* Required credentials (API keys, OAuth details) must be available

&#x20;

**Steps**

1. From the left navigation, click Integrations
2. Click Create Integration
3. Select the Integration Type (e.g., REST API, SaaS Connector)
4. Enter:
5. Integration Name
6. Description
7. Configure authentication:
8. API Key
9. OAuth
10. Token-based (as applicable)
11. Click Save

&#x20;

**What Happens Next**

* Integration is created in Active state
* Integration becomes available to:
* Connector Tasks
* Processes
* Activities

&#x20;

**Important Notes**

* Integrations are reusable assets
* Credentials are stored securely
* Integration definitions are version-controlled internally
