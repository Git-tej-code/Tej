# Add Process to a Package

**Who can do this**

* Platform Admin
* Power User (with deployment permission)

&#x20;

**Steps**

1. Open the target Package
2. Click Edit Package
3. Add or update:
4. Processes
5. Process versions
6. Save changes

&#x20;

**Alternative Method (From Process Canvas)**

1. Open a Process
2. Click Save As
3. Select target Package
4. Save

&#x20;

**What Happens Next**

* Package now references selected process versions
* Package version updates automatically

&#x20;

**Important Notes**

* You can choose custom versions per process
* Once locked, mappings cannot be changed
* Version consistency is enforced
