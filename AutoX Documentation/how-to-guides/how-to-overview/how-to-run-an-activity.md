# How to Run An Activity

**Who can do this**

* Platform Admin
* Power User

&#x20;

**Prerequisites**

* Activity must be Unlocked
* All required inputs must be configured

&#x20;

**Steps**

1. Open the Activity
2. Click Run
3. Provide input values (if prompted)
4. Start execution

&#x20;

**What Happens Next**

* Activity executes independently
* Execution logs are generated
* Results are visible in Process Logs and Agent Observer

&#x20;

**Important Notes**

* Running an activity is for testing only
* It does not deploy the activity
* Failures do not affect other versions
