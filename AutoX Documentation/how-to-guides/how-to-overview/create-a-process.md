# Create a Process

**Who can do this**

* Platform Admin
* Power User

Business Users may view processes but cannot create them.

&#x20;

**Prerequisites**

* Department must exist and be Active
* Function must exist and be Active
* At least one Package must be available (default package is auto-created)

&#x20;

**Steps**

1. Navigate to Processes from the left navigation
2. Click Create Process
3. Select:
4. Department
5. Function
6. Package
7. Enter:
8. Process Name
9. Description
10. Click Create

&#x20;

**What Happens Next**

* A new Process Version (v1.0) is created
* Process status is set to In Progress
* Process is editable on:
* WorkJam Board
* Develop (Canvas)

&#x20;

**Important Notes**

* Process names can repeat, but versions are unique
* Processes cannot be deleted
* Every edit creates a new version automatically
